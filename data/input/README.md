# Input Data

This directory contains data that has been prepared for processing or analysis.

## Guidelines

- Data that is ready to be consumed by scripts or models
- May include cleaned, filtered, or preprocessed data
- Can be derived from raw data or external sources
- Should be analysis-ready format

## File Organization

- Use clear, descriptive naming conventions
- Include processing date: `dataset_processed_YYYY-MM-DD.ext`
- Document any transformations applied from raw data
