# Output Data

This directory contains results from data processing, analysis, or model outputs.

## Guidelines

- Final results, reports, and generated datasets
- Model outputs, predictions, and analysis results
- Processed data ready for sharing or publication
- Should be clearly documented and reproducible

## File Organization

- Use descriptive naming with timestamps: `results_YYYY-MM-DD_HHMM.ext`
- Include version numbers for iterative analyses
- Document methodology and parameters used
- Include summary statistics or metadata files
